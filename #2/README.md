# Mobile Management App

This is a Node.js and Express application that provides a complete CRUD (Create, Read, Update, Delete) interface for managing a database of Mobile phones. It uses MongoDB for data storage and EJS for rendering the user interface.



## Features

- **Home Page**: Overview of the application.
- **Add New Mobile**: Insert new mobile phone records with details like Brand, Model, RAM, Storage, and Price.
- **View All Mobiles**: Display all the stored mobile phone records in a clean layout.
- **Edit Records**: Update the details of any existing mobile phone in the database.
- **Delete Records**: Remove any mobile phone record from the database.
- **Search Functionality**: Search for specific mobile phones from the database.

## Technologies Used

- **Backend**: Node.js, Express.js
- **Database**: MongoDB, Mongoose
- **View Engine**: EJS (Embedded JavaScript templates)
- **Styling**: CSS (Static assets served via Express)
